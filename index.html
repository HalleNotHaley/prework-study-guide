Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML elements work together to build the structure of a webpage and provide a framework that makes it easy for a user to navigate the webpage.
  • Opening and closing elements are usually represented with angle brackets.
  • Elements included within the hea are not visible when a user looks at a webpage.
  • All the elements that we want the user to see and interact with must be placed inside the body element.
  • !DOCTYPE is a declaration that tells the browser what document type it should expect when it is opened.
  • !DOCTYPE html is letting the browser know to expect HTML on the page.
  • !DOCTYPE is not case-sensitive. HTML elements are case-sensitve.
  • Metadata is data about our webpage that the browser uses to effectively serve our webpage to users.
  • Metadata is typically not seen by users, but it contains important information about accessibility, search engines, and viewpoint.
  • The title element defines the title for a webpage. This element is required and is extremely important for Search Engine Optimization (SEO).
  • The style element defines the CSS styles associated with a website.
  • The base element defines the base URL fpr a webpage.
  • The meta element defines metadata such as the character set, description, keywords, author, and viewpoint.
  • The script element embeds client-side scripts in the HTML document.
  • The meta element defines the charset, or character set. In the example webpage, it defines the charset as UTF-8, which allows us to use most characters and symbols on the page.
  • The div element creates division in the page.
  • The span element creates an inline division.
  • The words "element" and "tag" are used interchangeably in the web development world, but it is important to know the difference.
  • An element refers to the entire element from opening tag to closing tag, whereas a tag refers only to what appears inside the angle brackets.
  • The h1, h2, h3, h4, h5, and h6 elements represent the level of a heading a given text block represents.
  • The p element represents a paragraph or block of text.
  • The ul, ol, and li elements represent unordered lists, ordered lists, and list items.
  • The img element contains information about images that are displayed on the webpage.
  • The img element can contain various attributes.
  • The src attribute is the most important because it defines the location of the image.
  • The br element creates a line of empty space, or a line break, between two blocks of content.
  • The a element, which stands for anchor, creates links to the same webpage or other webpages.
  • The href attribute points to the URL for the link.

CSS

  • CSS stands for Cascading Style Sheets.
  • A link element creates the connection to the exteral file
  • The rel attribute specifies the relationship between the current document and the linked document resource.
  • The href attribute specifies the location (URL) of the external resource.
  • The href attribute is using a relative path. This means the location is derived from the current location.
  • To locate the style sheet from the current location, we must look in the assets folder, hence assets/style.css
  • Write new declarations for a new CSS rule by designating a class attribute as the selector.
  • Class attributes allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector.
  • The class attribute can be assigned to the element in the HTML, enabling us to freely choose which elements will be assigned the CSS rule.
  • The period preceding card designates this as a class selector. Any HTML element assigned to the class, card, will gain these property assignments from the CSS.
  • The first declaration addresses the width of the element at #% of the available width.
  • The margin property assigns #px to the top and bottom, and the auto value to the left and right.
  • Example: The border property assigns a border with a width of 5px that is solid and gray.
  • The box-shadow CSS property adds shadow effects around an element's frame.
  • A box shadow is described by X an Y offsets relative to the element, blur and spread radius, and color.
  • Example: the X offset is 5px and defines the width or extent of the shadow on the X axis (or to the right of the box); the Y offset is 10px and defines the extent on the Y axis (or just below the box).
  • In the example, an HTML hexadecimal code defines the color.
